Numerologia

Numerologia é um método que analisa nosso perfil a partir do significado de cada número. A partir da data de nascimento, da soma dos valores das letras do nome, utilizando um gráfico para distribuição dos números, podemos realizar uma analise capaz de direcionar nossos caminhos e promover uma bela base para nos tornarmos pessoas melhores, mais felizes e com alta auto estima.
